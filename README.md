# FoodTrends-Understanding-Customer-Preferences-Based-on-FB
This project aims to analyze food trend data to understand customer preferences based on factors such as cuisine, ratings, pricing, and ordering behavior.
The goal is to derive insights that help businesses make data-driven decisions in the food and restaurant industry.

**Objectives
Analyze customer food preferences
Identify popular cuisines and trends
Understand factors influencing customer choices
Build predictive or recommendation insights (if applicable)

**Tech Stack (Planned)
Power Bi
Python
Pandas
NumPy
Matplotlib / Seaborn
Machine Learning (Scikit-learn)
Jupyter Notebook

Repository initialized. Data collection and exploratory analysis will begin next.
